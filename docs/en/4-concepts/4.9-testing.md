### E2E Testing

#### Install Cypress App

```shell
bunx cypress install
```

#### Editing E2E Test Suite

```shell
cd apps/web
bunx cypress open
```

This command will open the cypress UI which allows for a live demonstration of the e2e test suit

#### Run E2E Test Suite

```shell
cd apps/web
bunx cypress run
```

This command will run the cypress test suite
